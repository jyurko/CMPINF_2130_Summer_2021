The grouped data frame allows us to SPLIT data based on variables, then APPLY functions/methods to the different groups, and then ultimately COMBINE all the results together. This is the split-apply-combine workflow.

Let's make a graphic in Seaborn where the x and y semantic variables (the aesthetics) are mapped to the Sepal variables and we use color, shape, and facets to denote the Species.

We want to use Seaborn figure level functions to enable easily creating the facets.

In [16]:
sns.relplot(data = iris, 
            x='Sepal.Length', y='Sepal.Width',
            hue = 'Species', style = 'Species',
            col ='Species', 
            alpha = 0.5, s=200)

plt.show()

The "geometric object" in Seaborn is represented essentially by the kind of plot.
